Posted on 2016-04-09 | In markdown |   |   次阅读

标题

标题的前加#，#越多字体越小，级数越小，会反映到目录中去例如：

#1级标题
##2级标题
###3级标题

列表写法

无序列表可以用*也可以用-号，注意空格

- 这是无序列表
    - 这是无序列表 
    - 这是无序列表
- 这是无序列表

列表

列表写法如下：

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

col 3 is right-aligned $1600
col 2 is centered $12
zebra stripes are neat $1
